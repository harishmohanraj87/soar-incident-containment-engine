# SOAR Incident Containment Engine

A Security Orchestration, Automation, and Response (SOAR) platform that automates security alert processing, threat intelligence enrichment, and incident containment actions.

## Project Goals

- Receive security alerts from SIEM systems
- Extract Indicators of Compromise (IOCs)
- Enrich alerts using threat intelligence sources
- Execute automated containment playbooks
- Provide incident visibility through a dashboard

## Tech Stack

- Python
- FastAPI
- GitHub
- VirusTotal
- AbuseIPDB
